## 基数変換
対象となる数の基数（桁上がりする基準の数でN進数のNにあたる）を異なる基数へ変換する。

- 例 \\(1\\) : 整数の基数変換 \\(10\\) → \\(2\\)

    \\((25)_{10}\\) の基数を \\(2\\) に変換する。<br>
    **変換元の基数**と**変換先の数**の剰余から導く。

    \\(25 \div 2 = 12 \\) 余り \\( 1 \\) <br />
    \\(12 \div 2 = 6 \\) 余り \\( 0 \\) <br />
    \\(6 \div 2 = 3 \\) 余り \\( 0 \\) <br />
    \\(3 \div 2 = 1 \\) 余り \\( 1 \\) <br />
    \\(1 \div 2 = 0 \\) 余り \\( 1 \\) <br />
    → 最後の剰余から順に並べて \\((11001)_{2}\\)

- 例 \\(2\\) : 整数の基数変換 \\(10\\) → \\(16\\)

    \\((555)_{10}\\) の基数を\\(15\\)に変換する。<br>
    **変換元の基数**と**変換先の数** の剰余から導く。

    \\(555 \div 16 =  34 \\) 余り \\( 11(B) \\) <br />
    \\(34 \div 16 = 2 \\) 余り \\( 2 \\) <br />
    \\(2 \div 16 = 0 \\) 余り \\( 2 \\) <br />
    → 最後の剰余から順に並べて \\((22B)_{16}\\)

- 例 \\(3\\) ： 小数の基数変換 \\(10\\) → \\(2\\)

    \\((0.75)_{10}\\) の基数を \\(2\\) に変換する。<br>
    **変換元の数**に**変換先の基数**をかけた整数部から導く。

    \\(0.75 \times 2 = 1.50\\) → 整数部 \\(1\\)<br />
    \\(0.50 \times 2 = 1.00\\) → 整数部 \\(1\\)<br />
    \\(0.00 \times 2 = 0.00\\) → 整数部 \\(0\\)<br />
    → 最後の整数部から順に並べて \\((0.11)_{2}\\)

- 例 \\(4\\) ： 小数の基数変換 \\(2\\) → \\(10\\)

    \\((110.01)_{2}\\) の基数を \\(10\\) に変換する。<br>
    **変換元の各桁**に**基数のべき乗**をかけた総和から導く。

    \\((110.01)_{2}\\)<br />

    \\(= 4 + 2 + 0\\) \\(+ 0 + 0.25\\) \\(= (6.25)_{10}\\)
