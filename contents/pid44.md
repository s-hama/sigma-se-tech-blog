## タイトル
応用情報技術 - 基礎覚書き：離散数学

## 目的
この記事では、応用情報技術者試験の出題分野である**離散数学**に関する覚書きを列挙する。

## 基数変換
対象となる数の基数（桁上がりする基準の数でN進数のNにあたる）を異なる基数へ変換する。

- 例 \\(1\\) : 整数の基数変換 \\(10\\) → \\(2\\)

    \\((25)_{10}\\) の基数を \\(2\\) に変換する。<br>
    **変換元の基数**と**変換先の数**の剰余から導く。

    \\(25 \div 2 = 12 \\) 余り \\( 1 \\) <br />
    \\(12 \div 2 = 6 \\) 余り \\( 0 \\) <br />
    \\(6 \div 2 = 3 \\) 余り \\( 0 \\) <br />
    \\(3 \div 2 = 1 \\) 余り \\( 1 \\) <br />
    \\(1 \div 2 = 0 \\) 余り \\( 1 \\) <br />
    → 最後の剰余から順に並べて \\((11001)_{2}\\)

- 例 \\(2\\) : 整数の基数変換 \\(10\\) → \\(16\\)

    \\((555)_{10}\\) の基数を\\(15\\)に変換する。<br>
    **変換元の基数**と**変換先の数** の剰余から導く。

    \\(555 \div 16 =  34 \\) 余り \\( 11(B) \\) <br />
    \\(34 \div 16 = 2 \\) 余り \\( 2 \\) <br />
    \\(2 \div 16 = 0 \\) 余り \\( 2 \\) <br />
    → 最後の剰余から順に並べて \\((22B)_{16}\\)

- 例 \\(3\\) ： 小数の基数変換 \\(10\\) → \\(2\\)

    \\((0.75)_{10}\\) の基数を \\(2\\) に変換する。<br>
    **変換元の数**に**変換先の基数**をかけた整数部から導く。

    \\(0.75 \times 2 = 1.50\\) → 整数部 \\(1\\)<br />
    \\(0.50 \times 2 = 1.00\\) → 整数部 \\(1\\)<br />
    \\(0.00 \times 2 = 0.00\\) → 整数部 \\(0\\)<br />
    → 最後の整数部から順に並べて \\((0.11)_{2}\\)

- 例 \\(4\\) ： 小数の基数変換 \\(2\\) → \\(10\\)

    \\((110.01)_{2}\\) の基数を \\(10\\) に変換する。<br>
    **変換元の各桁**に**基数のべき乗**をかけた総和から導く。

    \\((110.01)_{2}\\)<br />
    \\( = 1 \times 2^{2} + 1 \times 2^{1} \\(+ 0 \times 2^{0} + 0 \times 2^{-1}\\) \\(+ 1 \times 2^{-2}\\) <br />
    \\(= 4 + 2 + 0\\) \\(+ 0 + 0.25\\) \\(= (6.25)_{10}\\)


## 補数表現（負の表現）
- 補数表現<br>
    対象の数（決められた基数と桁数の条件下）に加算すると桁上がりする最小の数。

    - 例 \\(1\\) : \\(2\\) 桁の \\(10\\) 進数 \\((55)_{10}\\) の補数。

        \\(10^{2} - (55)_{10}\\) \\(= (45)_{10}\\)

    - 例 \\(2\\) : \\(8\\) 桁の \\(2\\) 進数 \\((00001101)_{2}\\) の補数。

        ※ \\(2\\)進数の場合、全桁（ビット）反転して1を加算する。<br>
        → \\((11110011)_{2}\\)

- 補数表現するメリット<br>
    減算を加算のみで表現できる。<br>

    - 例 \\(1\\) : \\((25)_{10} - (13)_{10}\\) を加算のみで表現する。

        \\((25)_{10} - (13)_{10}\\) \\(= (25)_{10} + ( - 13)_{10}\\)<br>
        ここで負の表現を補数表現で置き換えると、\\((25)_{10} + (87)_{10}\\) \\(= (112)_{10}\\)<br>
        よって、\\(2\\) 桁の範囲で見ると \\(12\\) となる。

    - 例 \\(2\\) : \\((25)_{10} - (14)_{10}\\) を \\(2\\)進数で加算のみで表現する。

        \\((25)_{10} - (14)_{10}\\) \\(= (00011001)_{2}\\) \\(- (00001110)_{2} \\)<br>
        ここで負の表現を補数表現で置き換えると、<br>
        \\((00011001)_{2}\\) \\(+ (11110010)_{2}\\) \\(= (100001011)_{2}\\)<br>
        よって、\(8\) 桁の範囲で見ると \\((00001011)_{2}\\) \\(= (11)_{10} \\) となる。

## 小数表現
- 固定小数点数表現<br>
    制限された桁数（ビット）内で小数点の位置を固定し数値を表現する。

- 浮動小数点数表現<br>
    数値に応じて、制限された桁数（ビット）内で**仮数部**と**指数部**に分類して数値を表現する。<br>
    → 固定小数点表現に比べ、より広義の数値表現ができる。

- 小数表現の正規化<br>
    **仮数部**と**指数部**に分けて表現を標準化すること。<br>
    ※ 仮数部と指数部を分ける基準（仮数部整数の大きさ）は、与えられた条件次第であることに注意。

    - 例 \\(1\\) : \\((0.00052)_{10}\\) を正規化する。

        \\((0.00052)_{10}\\) \\(= (5.2)_{10} \times 10^{-4}\\)  <br>
        → \\(5.2\\) が仮数部、\\(10^{-4}\\) が指数部  <br>
        \\((0.00052)_{10}\\) \\(= (0.52)_{10} \times 10^{-3}\\)  <br>
        → \\(5.2\\) が仮数部、\\(10^{-3}\\) が指数部など。

    - 例 \\(2\\) : \\((0.011)_{2}\\) を正規化する。

        \\((0.011)_{2}\\) \\(= (1.1)_{2} \times 2^{-2}\\)  <br>
        → \\(1.1\\) が仮数部、\\(2^{-2}\\) が指数部  <br>
        \\((0.011)_{2}\\) \\(= (0.11)_{2} \times 2^{-1}\\)  <br>
        → \\(0.11\\) が仮数部、\\(2^{-1}\\) が指数部など。  <br>
